## Run Locally

Clone the project

```bash
  git clone https://github.com/soydavidpaucar/react-typescript-github-profile-search.git
```

Go to the project directory

```bash
  cd react-typescript-github-profile-search
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn preview
```
